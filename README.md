# Stack Undo & Redo System (C++)

A C++ implementation of the Stack data structure demonstrating the Undo and Redo mechanism commonly used in text editors, design tools, and many modern applications.

### Features

- Stack-based Undo functionality
- Stack-based Redo functionality
- Action history tracking
- Navigation through previous and next states
- Console-based simulation

### Concepts Practiced

- Stack Data Structure
- LIFO (Last In, First Out)
- Undo/Redo Design Pattern
- State Management
- Data Structure Applications
- Object-Oriented Programming (OOP)

### Technologies

- C++
- Standard Template Library (STL)
- Console Application

## Purpose

This project was developed to demonstrate a practical real-world application of the Stack data structure by implementing an Undo and Redo system similar to those used in professional software applications.
